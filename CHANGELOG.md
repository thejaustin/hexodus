# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Advanced theming features (gradient, animated, texture themes)
- System inspection capabilities (based on LibChecker)
- Per-app theming features (based on DarQ)
- Gesture and interaction customization (based on TapTap)
- Media and notification customization (based on AmbientMusicMod)
- Backup and restore functionality
- Wallpaper-based theming features
- Advanced overlay management
- Font and icon management features
- System tuning capabilities
- Privacy and security management
- Network management features
- Performance optimization tools

### Changed
- Updated UI/UX to comply with Material 3 Extended (M3E) standards
- Enhanced security validation across all services
- Improved accessibility compliance
- Updated theming engine with enhanced color generation
- Refined Shizuku integration for better reliability

### Fixed
- Various UI/UX issues for better user experience
- Security vulnerabilities in input validation
- Accessibility issues with touch targets and contrast
- Performance issues with theme compilation

## [1.0.0] - 2026-01-29

### Added
- Initial release of Hexodus theming engine
- Core theming functionality with hex color support
- Shizuku integration for system-level operations
- Material You override for Samsung One UI 8
- High contrast theme injection
- Foldable device support for Z Flip 5
- Component-specific theming (status bar, navigation bar, etc.)
- Theme preview functionality
- Theme sharing capabilities
- Real-time theme application
